About the Algerian Forest Fires Dataset

The Algerian Forest Fires Dataset contains data from two regions of Algeria: Bejaia and Sidi Bel-abbes, covering the period from June 2012 to September 2012. It includes 122 instances for each region, totaling 244 instances. The dataset provides information on various weather and fire-related attributes.

Key Dataset Attributes

  • Date: Day, month ('june' to 'september'), year (2012)
  • Temp: Noon temperature (max) in Celsius degrees (22 to 42)
  • RH: Relative Humidity in % (21 to 90)
  • Ws: Wind speed in km/h (6 to 29)
  • Rain: Total rainfall in mm (0 to 16.8)
  • FFMC: Fine Fuel Moisture Code index from the FWI system (28.6 to 92.5)
  • DMC: Duff Moisture Code index from the FWI system (1.1 to 65.9)
  • DC: Drought Code index from the FWI system (7 to 220.4)
  • ISI: Initial Spread Index index from the FWI system (0 to 18.5)
  • BUI: Buildup Index index from the FWI system (1.1 to 68)
  • FWI: Fire Weather Index (FWI) Index (0 to 31.1)
  • Classes: Two classes, namely Fire and not Fire
